# Station Dashboard

A static, data-free operations dashboard and tool set. No accounts, no servers,
no personal data ships in this build — every report or schedule you load stays
in your own browser (localStorage / on-device only).

Tools included: the dashboard hub, the DSW (In-Area 12 / MDVS catcher), the
MasterList Builder, the QA Checklist, and the Schedule analyzer. The IBNO Coder
and Time-Off tools are hosted separately and linked from the dashboard.

Built from a private source repository and published with a deliberate deploy
step; this public repo contains only the audited static build.
